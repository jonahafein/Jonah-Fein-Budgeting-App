To use this app, follow these instructions:

1. Clone this repository

2. Create a Supabase Project
Go to: https://supabase.com
Sign in and click “New Project”
Fill in:
Project name → budgeting-app (or anything)
Database password → save this somewhere

Click Create and wait ~1–2 minutes
Get API Credentials
Go to: Project Settings → API

Copy:
Project URL
Publishable (anon) key
Add Config File

Create this file:
backend/config.py

Add:

supabase_url = "https://YOUR_PROJECT_ID.supabase.co"
publishable_key_supabase = "YOUR_ANON_KEY"
Also add your Groq API key:
groq_api_key = "YOUR_GROQ_KEY"

Get a Groq key here:
https://console.groq.com/keys

3. Create Database Tables
Go to Supabase → SQL Editor
Paste and run the SQL script from sql_table_creation.sql

Disable Row Level Security:
In Supabase SQL Editor, run:

ALTER TABLE users DISABLE ROW LEVEL SECURITY;
ALTER TABLE dashboard DISABLE ROW LEVEL SECURITY;
ALTER TABLE debt DISABLE ROW LEVEL SECURITY;
ALTER TABLE expenses DISABLE ROW LEVEL SECURITY;
ALTER TABLE goals DISABLE ROW LEVEL SECURITY;
ALTER TABLE home DISABLE ROW LEVEL SECURITY;
ALTER TABLE income DISABLE ROW LEVEL SECURITY;
ALTER TABLE non_home_assets DISABLE ROW LEVEL SECURITY;

4. Run Setup Script:
chmod +x setup.sh
./setup.sh
source venv/bin/activate

5. Run the App:
cd app
streamlit run Login.py
